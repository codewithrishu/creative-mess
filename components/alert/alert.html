Simple Alert

Simple alert can be created by adding alert class, followed by color varient

This is Primary alert!
This is Error alert!
This is Success alert!
This is Warning alert! 
                                             
                                            <div class="alert primary">             
                                                This is Primary alert!
                                            </div>
                
                                            <div class="alert error">
                                                This is Error alert!
                                            </div>
                
                                            <div class="alert success">
                                                This is Success alert!
                                            </div>
                
                                            <div class="alert warning">
                                                This is Warning alert!
                                            </div>

Outline Alert

Outline alert can be created by adding alert class, followed by outine varient

This is Primary alert!
This is Error alert!
This is Success alert!
This is Warning alert! 
                                             
                                            <div class="alert outline-primary">
                                                This is Primary alert!
                                            </div>
                                            <div class="alert outline-error">
                                                This is Error alert!
                                            </div>
                                            <div class="alert outline-success">
                                                This is Success alert!
                                            </div>
                                            <div class="alert outline-warning">
                                                This is Warning alert!
                                            </div>
