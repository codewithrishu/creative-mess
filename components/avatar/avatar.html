Avatars

Avatars are images that users can set as their profile picture.they're rounded and square shaped.

Profile Avatar

Add the 'avatar' class to img tag to simply create profile avatar 

                                
                        <img class="avatar" src="https://url-of-image" alt="">

Size Avatar

There are 4 additional sizes available, xs | sm | lg | xl

                                
                        <img class="avatar xs" src="https://url-of-image" alt="">
                        <img class="avatar sm" src="https://url-of-image" alt=""> 
                        <img class="avatar lg" src="https://url-of-image" alt=""> 
                        <img class="avatar xl" src="https://url-of-image" alt="">

Square Avatar

Add the 'avatar' class to img tag to simply create square avatar

                                
                          <img class="square avatar xs" src="https://url-of-image" alt="">
                          <img class="square avatar sm" src="https://url-of-image" alt=""> 
                          <img class="square avatar lg" src="https://url-of-image" alt=""> 
                          <img class="square avatar xl" src="https://url-of-image" alt="">
