Primary Buttons

Primary button can be created by adding button class followed by color varient

                                     
                                    <button class="button primary">PRIMARY</button>
                
                                    <button class="button error">ERROR</button>
                
                                    <button class="button success">SUCCESS</button>
                
                                    <button class="button warning">WARNING</button>

Outline Buttons

Outline button can be created by adding outline-button class followed by outline-color varient

                                     
                                    <button class="outline-button outline-primary">PRIMARY</button>    
                
                                    <button class="outline-button outline-error">ERROR</button>
                
                                    <button class="outline-button outline-success">SUCCESS</button>
                                    
                                    <button class="outline-button outline-warning">WARNING</button>

Link Buttons

Link button can be created by adding button class followed by button-link class

                                     
                                    <button class="button button-link primary">
                                        <a href="">LINK</a>
                                    </button>
