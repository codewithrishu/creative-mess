Headings

headings are defined with the H1 to H6 tags. H1 defines the most important heading. H6 defines the least important heading.

h1.This is Heading

h2.This is Heading

h3.This is Heading

h4.This is Heading

h5.This is Heading
h6.This is Heading
                    
                        <h1>h1.This is Heading</h1>
                        <h2>h2.This is Heading</h2>
                        <h3>h3.This is Heading</h3>
                        <h4>h4.This is Heading</h4>
                        <h5>h5.This is Heading</h5>
                        <h6>h6.This is Heading</h6>

Paragraph

you can add different sizes of paragraph

This is large size paragraph

This is medium size paragraph

This is small size paragraph

                     
                        <p class="lg-para">This is large size paragraph</p>
                        <p class="md-para">This is medium size paragraph</p>
                        <p class="sm-para">This is small size paragraph</p>

Small Text

Small Text can e added where you dont want much of users attention

This is small text 
                     
                        <small>This is small text</small>

Coloured Text

you can add colored text by simply adding particular colour-text class. Grey | Red | Green

This is grey text This is green text This is red text 
                     
                        <small class="grey-text">This is grey text</small>
                        <small class="green-text">This is green text</small>
                        <small class="red-text">This is red text</small>

Aligned Text

Different alignment can be added to your text

Left Aligned Text

Center Aligned Text

Right Aligned Text

                    
                        <p class="left-align-text">Left Aligned Text</p>
                        <p class="center-align-text">Center Aligned Text</p>
                        <p class="right-align-text">Right Aligned Text</p>
